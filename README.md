# Data Analytics Using Python

## Project Overview

This repository demonstrates the implementation of various data analytics techniques using Python and machine learning libraries. The project covers the complete analytics workflow from data loading and preprocessing to model development and evaluation.

## Implemented Modules

1. Data Loading and Exploration (EDA)
2. Data Preprocessing and Cleaning
3. Data Visualization Techniques
4. Statistical Analysis
5. Regression Analysis
6. Classification Techniques
7. Clustering Algorithms
8. Dimensionality Reduction using PCA
9. Time Series Analysis using ARIMA
10. Mini Project: End-to-End Data Analytics Pipeline

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* SciPy
* Statsmodels

## Datasets

* Airline Passenger Dataset
* Heart Disease Dataset
* Sample datasets for analysis tasks

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis
* Machine learning model implementation
* Data visualization
* Statistical analysis
* Forecasting and prediction
